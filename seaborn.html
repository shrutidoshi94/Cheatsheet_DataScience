Introduction to Seaborn

Seaborn is a declarative statistical graphics library built on top of Matplotlib. It integrates closely with Pandas DataFrames and provides a high-level interface for drawing attractive graphics. Understanding its function hierarchy is key to mastering the library.

Relational Plots (relplot)

Use when you want to show how variables relate to each other (e.g., correlations or trends over time). Includes scatterplot and lineplot.

Distribution Plots (displot)

Use when you need to understand the underlying frequency, density, or shape of a dataset. Includes histplot, kdeplot, and ecdfplot.

Categorical Plots (catplot)

Use when you want to compare the distribution or central tendency of a numeric variable across discrete categorical groups (e.g., boxplot, violinplot, barplot).

Matrix Plots

Use for visualizing 2D array data or correlation matrices. Functions like heatmap use color gradients to represent values, making patterns instantly visible.
